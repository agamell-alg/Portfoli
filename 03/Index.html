Tipos de formatos para web

Uno de las principales decisiones a la hora de incluir gráficos en tu web será elegir el formato correcto para cada tipo de imagen.

Cuando hablamos de formatos de imagen en la web tenemos que limitarnos a 3:


1. JPEG(Joint Photographic Experts Group)

Es un método para comprimir las imágenes y también un formato de archivo. JPG es un tipo de archivo que fue desarrollado por el Joint Photographic Experts Group (JPEG) para que fuese el estándar para las imágenes hechas por fotógrafos.

Características
  1. Formato gráfico de mapa bits.
  2. Soporte de color verdadero, también conocido por su homónimo inglés true color (24 Bits).
Ventajas
  • Formato comprimido.
  • Puede ser comprimido hasta 100 veces.
  • Posee una amplia gama de colores.
  • Mantiene una buena calidad tras la compresión.
  • Ideal para mostrar fotografías en la web dado que no suele ocupar mucho espacio.
Desventajas
  • A pesar que mantiene la calidad, cuenta con pérdidas.
  • No soporta animaciones ni fondos transparentes.
  • No es adecuado para ilustraciones, dibujos, ni textos.
  • Genera pixelación aún configurándola con la mayor calidad.
  • No es un formato multi-página.

2. PNG (Portable Network Graphics)

Formato desarrollado en buena parte para solventar las deficiencias del formato GIF. Permite almacenar imágenes con una mayor profundidad de contraste.

Características
  1. Permite almacenar imágenes en blanco y negro.
  2. Utiliza una paleta de 256 colores.
Ventajas
  • Formato comprimido.
  • Sin pérdidas.
  • Soporta millones de colores.
  • Soporta fondos transparentes y semitransparentes.
  • Ideal para el desarrollo de logotipos, botones y otro tipo de imágenes que necesiten conservar una buena resolución.
Desventajas
  • No soporta animaciones.
  • No es un formato multi-página.
  • Imágenes con muchos elementos y colores pueden pesar mucho más que su equivalente en jpg.
  • Incompatibilidad con navegadores antiguos.
  • No admite modo CMYK.

3. GIF (Graphics Interchange Format)

Se utilizan, normalmente, para crear animaciones o imágenes en movimiento.

Características
  1. Soporta como máximo 256 colores.
  2. Tiene un formato ligero.
Ventajas
  • Soporta animaciones y fondos transparentes.
  • Sin pérdidas.
  • Tamaño reducido.
  • Ayuda a construir una imagen moderna en los negocios.
  • Muy usado para imágenes pequeñas.
Desventajas
  • No es un formato multi-página.
  • Baja calidad respecto a otros formatos.
  • Limitada profundidad de color (8 bits).

4. SVG (Scalable Vector Graphics)

Es un formato vectorial poco conocido pero muy útil para su uso online por su flexibilidad y por la capacidad de ofrecer gráficos con calidad.

Características
  1. Se almacena en ficheros de texto plano
  2. Es un formato vectorial
Ventajas
  • Tamaño pequeño.
  • Flexibilidad.
  • Escalabilidad sin cambiar la calidad de la imagen.
  • Es posible visualizar los contenidos de un archivo SVG en cualquier navegador
Desventajas
  • algunos navegadores aún no admiten este tipo de imagen.
  • No admite color CMYK.
